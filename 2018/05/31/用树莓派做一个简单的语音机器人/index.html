用树莓派做一个简单的语音机器人

设备及环境

树莓派3B

官方系统Raspbian（2017-04-10-raspbian-jessie）

免驱动的麦克风

小音响（也可以用耳机代替）

前期准备

在这个过程中，我们一共需要用到3个API，分别是百度语音里的语音识别和语音合成，还有图灵机器人。我们要分别注册账号，以便获取免费API的使用权。

1、注册百度云

注册百度云以便获得程序中需要用到的API Key和Secret Key，下面是获取的方法：

进入百度云的官网，注册后登陆，在主界面已开通的服务中选择百度语音

然后点创建应用

根据自己的需求按照提示完善信息

最后点立即创建即可，然后会获得这些信息，后面需要用到

2、注册图灵机器人

进入官网http://www.tuling123.com，注册并登陆，点击创建机器人

图5

从何获得apikey，这个后面也要用到

3、调整树莓派的音频输出

在终端执行

sudo raspi-config

然后选择7（Advanced Options ），然后选A4（Audio），进入选项后选择 Forece 3.5mm(‘headphone’) jack。确定然后保存退出重启系统。

4、安装播放器

sudo apt-get update

sudo apt-get upgrade

sudo apt-get install mpg123

sudo apt-get install omxplayer

5、麦克风测试

sudo arecord -D "plughw:1,0" -d 5 temp.wav

-D 参数：选择设备， 外部设备是 plughw:1,0，内部设备是 plughw:0,0， 树莓派本身并没有录音模块，所以没有内部设备。
-d 5参数：录制时间为5 秒，如果不加这个参数就会一直录音，直到按下 ctrl+c 才会停止。
temp.wav 是生成的文件名字。
存储位置就是当前所在目录下，当录制完成后，可以使用 ls 查看当前目录下是否保存了这样一个音频文件。

omxplayer -o local temp.wav

音频开始播放，在播放完成后 ，自动退出

6、切换python版本

树莓派中原始默认使用的是python2.7的版本，代码中有些库是用到的python3版本，所以需要进行切换树莓派上的python版本。这里默认树莓派上已经有了python3.5（如果没有可以自行安装，很简单，这里不再赘述）。下面开始切换版本：

  • 开始之前拷贝是很有必要的，要形成备份的习惯

    sudo cp /usr/bin/python /usr/bin/python_copy

  • 删除以前的这个2.7

    sudo rm /usr/bin/python

  • 重新指定新的链接

​ sudo cp /usr/bin/python3.5 /usr/bin/python

注意：有的情况python3.5文件在 /usr/local/bin/中，要根据自己的情况适当修改。

这里就完成的版本的切换，如果想切换回来，可以用同样的方法：

  • sudo rm /usr/bin/python
  • sudo cp /usr/bin/python_copy /usr/bin/python

获取签名

如果要调用百度语音API，还需要获得签名，需要使用创建应用所分配到的AppID、API Key及Secret Key，进行Access Token（用户身份验证和授权的凭证）的生成。

下面是python代码获取方式：

import urllib.request
import json
import base64
import sys

def get_access_token():
url = "https://openapi.baidu.com/oauth/2.0/token"
grant_type = "client_credentials"
client_id = "填写APIKey"
client_secret = "填写Secret Key"

url = url + "?" + "grant_type=" + grant_type + "&" + "client_id=" + client_id + "&" + "client_secret=" + client_secret

resp = urllib.request.urlopen(url).read()
data = json.loads(resp.decode("utf-8"))
return data["access_token"]

get_access_token()

Access Token的有效期为30天（以秒为单位），请集成时注意在程序中定期请求新的token。生成token后记得记下来，后面我们要用到。

具体也可参考http://ai.baidu.com/docs/#/Begin/top

python工程

要实现整个语音交互，首先就需要把需要识别的语音数据、语音信息、token等发送给百度的语音识别服务器，然后就能获取到对应的文字。

import urllib.request
import json
import base64
import sys

import urllib

import requests

def baidu_asr(data, id, token):
speech_data = base64.b64encode(data).decode("utf-8")
speech_length = len(data)

post_data = {
        "format" : "wav",
        "rate" : 16000,
        "channel" : 1,
        "cuid" : id,
        "token" : token,
        "speech" : speech_data,
        "len" : speech_length
}

url = "http://vop.baidu.com/server_api"
json_data = json.dumps(post_data).encode("utf-8")
json_length = len(json_data)
#print(json_data)

req = urllib.request.Request(url, data = json_data)
req.add_header("Content-Type", "application/json")
req.add_header("Content-Length", json_length)

print("asr start request\n")
resp = urllib.request.urlopen(req)
print("asr finish request\n")
resp = resp.read()
resp_data = json.loads(resp.decode("utf-8"))
if resp_data["err_no"] == 0:
    return resp_data["result"]
else:
    print(resp_data)
    return None

def asr_main(filename):
f = open(filename, "rb")
audio_data = f.read()
f.close()

#token = get_access_token()
token = "填写前面获取的token"
uuid = "填写前面获取的AppID"
resp = baidu_asr(audio_data, uuid, token)
print(resp[0])
return resp[0]

转换为相应的文字后，我们就需要考虑如何让树莓派实现对语音的回复，所以这里用到了图灵机器人，通过调用图灵机器人来实现树莓派的回复思考。

def Tuling(words):

Tuling_API_KEY = "填写前面获取的图灵机器人API_Key"

body = {"key":Tuling_API_KEY,"info":words.encode("utf-8")}

url = "http://www.tuling123.com/openapi/api"
r = requests.post(url,data=body,verify=True)

if r:
    date = json.loads(r.text)
    print(date["text"])
    return date["text"]
else:
    return None

最后则需要把回复的文字转化为语音说出来，即用到了语音合成，详细设置可以参考http://yuyin.baidu.com/docs/tts/136

def baidu_tts_by_post(data, id, token):

post_data = {
        "tex" : data,
        "lan" : "zh",
        "ctp" : 1,
        "cuid" : id,
        "tok" : token,
}

url = "http://tsn.baidu.com/text2audio"
post_data = urllib.parse.urlencode(post_data).encode('utf-8')
#print(post_data)
req = urllib.request.Request(url, data = post_data)

print("tts start request")
resp = urllib.request.urlopen(req)
print("tts finish request")
resp = resp.read()
return resp

def tts_main(filename, words):
token = "同前面的token"
text = urllib.parse.quote(words)
uuid = “11325809”
resp = baidu_tts_by_post(text, uuid, token)

f = open("test.mp3", "wb")
f.write(resp)
f.close()

最后整合到一起即可

words = asr_main("test.wav")
new_words = Tuling(words)
tts_main("test.mp3", new_words)

在写完这个python后，我们命名为main.py，然后写一个脚本如下：

\#! /bin/bash

arecord -D "plughw:1" -f S16_LE -r 16000 test.wav
python3 main.py
mpg123 test.mp3

现在就可以跟机器人对话了，运行脚本，对着麦克风说话，按ctri+c结束，然后就可以等待机器人的回话了哦！
