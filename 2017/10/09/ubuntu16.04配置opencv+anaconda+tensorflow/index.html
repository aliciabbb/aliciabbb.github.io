ubuntu16.04配置opencv+anaconda+tensorflow

注意：本台主机已经配置好cuda8.0和cudnn5.1

opencv3.2.0的安装

一 、安装依赖项

$ sudo apt-get install build-essential
$ sudo apt-get install cmake git libgtk2.0-dev pkg-config libavcodec-dev libavformat-dev libswscale-dev
$ sudo apt-get install python-dev python-numpy libtbb2 libtbb-dev libjpeg-dev libpng-dev libtiff-dev libjasper-dev libdc1394-22-dev # 处理图像所需的包
$ sudo apt-get install libavcodec-dev libavformat-dev libswscale-dev libv4l-dev liblapacke-dev
$ sudo apt-get install libxvidcore-dev libx264-dev # 处理视频所需的包
$ sudo apt-get install libatlas-base-dev gfortran # 优化opencv功能
$ sudo apt-get install ffmpeg

二 、解压下载好的压缩包

执行以下指令
$ cd opencv-3.2.0
$ mkdir build
$ cd build
$ cmake -D CMAKE_BUILD_TYPE=build -D CMAKE_INSTALL_PREFIX=/usr/local -D WITH_TBB=ON -D BUILD_NEW_PYTHON_SUPPORT=ON -D WITH_V4L=ON -D INSTALL_C_EXAMPLES=ON -D INSTALL_PYTHON_EXAMPLES=ON -D BUILD_EXAMPLES=ON -D WITH_QT=ON -D WITH_OPENGL=ON -D OPENCV_EXTRA_MODULES_PATH=~/opencv_contrib-3.2.0/modules -D WITH_CUDA=ON -D WITH_CUBLAS=ON -D DCUDA_NVCC_FLAGS="-D_FORCE_INLINES" -D CUDA_ARCH_BIN="8.0" -D CUDA_ARCH_PTX="" -D CUDA_FAST_MATH=ON -D WITH_GTK=ON -D PYTHON_EXCUTABLE=/usr/local/lib/python -D CUDA_GENERATION=Kepler ..
$ make -j8
$ sudo make install
$ sudo /bin/bash -c 'echo "/home/fanzong/anaconda2/envs/tensorflow/lib" > /etc/ld.so.conf.d/opencv.conf'
$ sudo ldconfig

PYTHON_EXCUTABLE：指定python路径
OPENCV_EXTRA_MODULES_PATH: 指定opencv所需模块的路径，就是之前我们所说的contrib

坑1：python导入cv2 出现如下错误：
Traceback (most recent call last):
File ““, line 1, in
ImportError: /usr/local/lib/libopencv_freetype.so.3.2: undefined symbol: hb_shape

解决1：

~/opencv-3.2.0/build$ sed -i 's/​${freetype2_LIBRARIES} ${harfbuzz_LIBRARIES}/​${FREETYPE_LIBRARIES} ${HARFBUZZ_LIBRARIES}/g'

~/opencv_contrib-3.2.0/modules/freetype/CMakeLists.txt

然后重新编译！！！

坑2：要根据是否有gpu来配置make变量，上面是有gpu版本的

anaconda的安装

一、下载安装包

地址：https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/要根据自己想使用的python版本下载。

鄙人选择了此版本 Anaconda2-5.1.0-linux-x86_64.sh

二、安装anaconda

进入上面下载的脚本目录文件，运行脚本即可。

全程只需要点yes和回车。

三、配置环境变量

sudo gedit ~/.bashrc

打开后，在最后一行加上：export PATH=”/home/username/anaconda2/bin:$PATH”

四、重启系统

tensorflow的安装

一、添加anaconda的TUNA镜像

conda config -add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

二、建立一个conda计算机环境

conda create -n tensorflow python=2.7

三、激活环境并使用conda安装tensorflow

source activate tensorflow

pip install --ignore-installed --upgrade https://storage.googleapis.com/tensorflow/linux/gpu/tensorflow_gpu-0.12.0rc0-cp27-none-linux_x86_64.whl

大坑：执行第三步的时候一直出现各种各样的错误，一直找不到解决方案。但是过了一晚上就好了，猜测是网速原因。

安装成功后，在终端输入python，然后import cv2失败，因为装了anaconda，所以会失败，这时执行：

pip install opencv-python

即可。

想要测试tensorflow，则需要在激活的conda环境中测试。
